# Cloud CMS

Cloud CMS is a content management system and is used to manage the following content:

  • News
  • Courses
  • Research Seminars
  • Fellowships
  • Events Calendar
  • Sleep and Breathing
  • Etc.

And it also allows to:

  • Manage the content of the ERS Connect app
  • Send App notifications
